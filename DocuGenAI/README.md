# DocuGenAI - Notebook Version

This folder contains the simple, all-in-one Jupyter notebook implementation of DocuGenAI.

## Files
- `docugenai-demo-a4.ipynb` - Main demo notebook
- `requirements.txt` - Dependencies for notebook
- `.env.example` - API key configuration template

## Quick Start
1. Copy `.env.example` to `.env` and add your Gemini API key
2. Install dependencies: `pip install -r requirements.txt`
3. Run the notebook: `jupyter notebook docugenai-demo-a4.ipynb`
